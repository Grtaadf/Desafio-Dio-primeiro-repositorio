# Comandos básicos para um bom desempenho terminal

**GUI x CLI**

- A maioria dos sistemas operacionais possuem programas que tem uma interface gráfica, ou seja, a forma que o usuário interage com esse programas e de forma gráfica, pode arrastar, clicar dentre outros.

O Git não tem uma interface gráfica, a forma de interagir com o Git é por linha de comando.
